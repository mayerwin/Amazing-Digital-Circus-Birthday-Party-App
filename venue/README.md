# `venue/` — your party space (optional, private)

Drop **photos and a floor plan of your party space** in this folder to help you
plan the *physical* side of the adventure. Nothing here is required to run the
app — it's purely a planning aid for you (and for an AI assistant if you use one
to help organise the day).

Everything inside `venue/` is **gitignored** (except this README), so your
private home photos are **never committed or shared**.

## What's useful to add
- A quick **floor plan / sketch** of the rooms and outdoor area (hand-drawn is fine).
- **Photos** of each space you'll use as an adventure "station".
- Notes on the route the kids will walk.

## How the app uses the physical space
The adventure moves the children through four "worlds" plus a finale. Map each
to a real spot in your home/garden, for example:

| Adventure beat            | A good real-world spot                                  |
|---------------------------|---------------------------------------------------------|
| Candy Canyon (catch game) | Garden / terrace with room to run                       |
| Mildenhall Manor (dark)   | A room you can darken — hide 15 glow stars              |
| "Don't Get Abstracted"    | Open space for a tag game                                |
| Fast-Food world (burger)  | A table to assemble the prop burger                      |
| The Exit Door (finale)    | A door the kids can dramatically push open               |
| Hidden treasure           | Anywhere you can stash the goodie bags                   |
| Cake time                 | Kitchen / living area                                    |

Hide the four Gloinks (one per world) and the treasure **before** the kids
arrive, then drive the adventure from Bubble's Guide. See the main
[README](../README.md) and `Nora-Circus-Party-Guide.html` for the full run sheet.

> Tip: generic spots like *terrace, pool, slide, guest bedroom, living room* work
> great — adapt to whatever your home offers.
